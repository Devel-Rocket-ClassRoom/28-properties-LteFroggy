# Student

학생의 성적을 관리하는 `Student` 클래스를 작성하세요.

**요구사항**

1. `Name` 속성: 학생 이름 (읽기 전용, 생성 시에만 설정)
2. `KoreanScore`, `EnglishScore`, `MathScore` 속성: 각 과목 점수
   - 0 ~ 100 범위만 허용 (범위 밖 값은 0 또는 100으로 조정)
3. `TotalScore` 속성: 총점 (읽기 전용, 계산된 값)
4. `Average` 속성: 평균 (읽기 전용, 계산된 값, double 타입)
5. `Grade` 속성: 학점 (읽기 전용, 계산된 값)
   - 평균 90 이상: "A"
   - 평균 80 이상: "B"
   - 평균 70 이상: "C"
   - 평균 60 이상: "D"
   - 평균 60 미만: "F"

**테스트 시나리오**

- 개체 이니셜라이저로 여러 학생 생성
- 성적 범위 초과 테스트 (110점, -10점 등)
- 각 학생의 성적표 출력

## 예상 실행 결과

```
=== 성적표: 김철수 ===
국어: 85, 영어: 90, 수학: 95
총점: 270
평균: 90.00
학점: A

=== 성적표: 이영희 ===
국어: 75, 영어: 80, 수학: 70
총점: 225
평균: 75.00
학점: C

=== 성적표: 박민수 ===
국어: 100, 영어: 100, 수학: 0
총점: 200
평균: 66.67
학점: D
```

> 참고: 박민수의 경우 입력값이 110, 120, -10이었으나 범위 조정으로 100, 100, 0이 됨
