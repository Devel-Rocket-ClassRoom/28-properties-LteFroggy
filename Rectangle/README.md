# Rectangle

사각형을 표현하는 `Rectangle` 클래스를 작성하세요.

**요구사항**

1. `Width` 속성: 가로 길이 (자동 속성, 기본값 1)
2. `Height` 속성: 세로 길이 (자동 속성, 기본값 1)
3. `Area` 속성: 넓이 (읽기 전용, 계산된 값 = Width x Height)
4. `Perimeter` 속성: 둘레 (읽기 전용, 계산된 값 = 2 x (Width + Height))
5. `IsSquare` 속성: 정사각형 여부 (읽기 전용, Width == Height이면 true)

**테스트 시나리오**

- 개체 이니셜라이저를 사용하여 여러 사각형 생성
- 각 사각형의 정보 출력 (가로, 세로, 넓이, 둘레, 정사각형 여부)

## 예상 실행 결과

```
=== 사각형 1 ===
가로: 10, 세로: 5
넓이: 50
둘레: 30
정사각형: False

=== 사각형 2 ===
가로: 7, 세로: 7
넓이: 49
둘레: 28
정사각형: True

=== 기본 사각형 ===
가로: 1, 세로: 1
넓이: 1
둘레: 4
정사각형: True
```
